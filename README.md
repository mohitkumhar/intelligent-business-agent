# Beta AI SaaS Web App - Frontend

## Setup & Run

1.  **Install dependencies**
    ```bash
    pnpm install
    ```

2.  **Environment Variables**
    Copy `.env.example` to `.env` and fill in the required values.
    ```bash
    cp .env.example .env
    ```

3.  **Start Local Development Server**
    ```bash
    pnpm dev
    ```

4.  **Build for Production**
    ```bash
    pnpm build
    ```

5.  **Run Tests**
    ```bash
    pnpm test
    ```

## Requirements
*   Node.js >= 18.0.0
*   pnpm >= 8
